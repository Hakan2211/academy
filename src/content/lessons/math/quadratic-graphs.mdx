<Step title="From line to curve" kind="explain">

A linear equation draws a straight line. Add an **x² term** and the graph bends into a **parabola** — the U-shaped (or ∩-shaped) curve you met in the Quadratics world, now seen as a *function*: for every input x, y = ax² + bx + c gives an output.

</Step>

</Step>

<Step title="Reading the key features off the graph" kind="explain">

A quadratic graph is defined by a few landmarks:

- **Direction:** a > 0 opens up (a minimum), a < 0 opens down (a maximum).
- **y-intercept:** the value of c (set x = 0).
- **x-intercepts (roots):** where y = 0 — solve ax² + bx + c = 0.
- **Vertex:** the turning point, on the line of symmetry x = −b/2a.

</Step>

<Step title="Sketching one quickly" kind="explain">

To sketch y = x² − 2x − 3 without plotting dozens of points:

1. Opens **up** (a = 1 > 0).
2. y-intercept at **−3**.
3. Roots: (x − 3)(x + 1) = 0 → x = 3 and x = −1.
4. Vertex on x = −b/2a = 1, giving y = −4, so (1, −4).

Plot those five facts, draw a smooth symmetric U, done. The symmetry means you get the left half free from the right.

</Step>

<Step title="Recap" kind="recap">

- A quadratic function y = ax² + bx + c graphs a **parabola**.
- Key features: **direction** (sign of a), **y-intercept** (c), **roots** (y = 0), **vertex** (x = −b/2a).
- The curve is **symmetric** about the vertical line through its vertex.
- Five landmarks are enough to sketch it.

</Step>
