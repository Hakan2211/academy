<Lesson>

<Step title="Where are you?" kind="explain">

Before we can talk about *motion*, we need to talk about **position** — where something is. And position only makes sense **relative to something else**.

You're "sitting still" reading this… but you're also hurtling around the Sun at 30 km/s. Both are true; they just use different **reference points**. Pick a reference point, and **motion is simply a change of position** over time.

</Step>

<Step title="Distance vs displacement" kind="explain">

Here's a subtlety that trips people up. Watch this object walk **8 m forward**, then **3 m back**:

<DistanceTrack />

- **Distance** is the *total path* it travelled: 8 + 3 = **11 m**.
- **Displacement** is the *net change* in position: it ended up **5 m** from where it started — and the **direction** matters.

Distance is just a number. Displacement is a number **with a direction**.

</Step>

<Step title="Two kinds of quantity" kind="explain">

That difference shows up all over physics. Some quantities are pure amounts; others come with a direction attached.

- **Scalars** — size only: distance, time, mass, temperature.
- **Vectors** — size **and** direction: displacement, velocity, force.

We can write displacement as the finish position minus the start:

<Formula tex="\Delta x = x_{\text{final}} - x_{\text{initial}}" block />

</Step>

<Step title="Recap" kind="recap">

- **Position** is location relative to a reference point; **motion** is changing position.
- **Distance** = total path travelled (a scalar).
- **Displacement** = net change in position, *with direction* (a vector).
- **Scalars** have size only; **vectors** have size and direction.

<Formula tex="\Delta x = x_{\text{final}} - x_{\text{initial}}" block />

</Step>
