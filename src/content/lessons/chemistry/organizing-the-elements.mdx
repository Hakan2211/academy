<Step title="A hundred-odd ingredients, one chart" kind="explain">

By the 1860s chemists had discovered dozens of elements but no order — just a long, confusing list. Then the Russian chemist **Dmitri Mendeleev** noticed something: if you line the elements up by increasing mass, their **properties repeat at regular intervals**.

He arranged them into rows so that elements with similar behaviour fell into the same column. The result was the **periodic table** — the single most useful chart in all of science.

</Step>

<Step title="Mendeleev's bold gaps" kind="explain">

Mendeleev's genius wasn't just sorting — it was *trusting the pattern*. Where no known element fit a slot, he left a **gap** and predicted the missing element's properties from its neighbours.

When gallium and germanium were later discovered with almost exactly the predicted properties, the table went from a clever filing system to a profound law of nature.

<Callout type="info" title="From mass to atomic number">
Mendeleev ordered by atomic mass, which mostly worked but occasionally misfired. We now order by **atomic number** (proton count, Z). That fix made every anomaly vanish — the true periodic pattern is set by protons, not mass.
</Callout>

</Step>

<Step title="The periodic law" kind="explain">

The big idea is the **periodic law**: *when elements are arranged by atomic number, their properties recur periodically.*

Why? Because properties are set by **valence electrons**, and the valence count resets and climbs again with each new row. Sodium starts a new row with one outer electron, just like lithium before it — so they behave alike. The table is really a map of repeating electron arrangements.

</Step>

<Step title="Recap" kind="recap">

- **Mendeleev** arranged the elements so that repeating properties lined up in columns — the **periodic table**.
- He left **gaps** and predicted missing elements, later confirmed — proving it was a law of nature.
- We now order by **atomic number** (Z), not mass.
- The **periodic law**: properties recur periodically because **valence electrons** reset each row.
