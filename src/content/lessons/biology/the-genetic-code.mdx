<Lesson>

<Step title="From letters to meaning" kind="explain">

DNA is a string of four letters — A, T, C, G. But a gene's real job is to specify a **protein**, which is a chain of **amino acids**. So somewhere there must be a code that translates the four-letter language of DNA into the twenty-letter language of amino acids.

That translation table is the **genetic code**, and it's one of the most elegant systems in biology.

</Step>

<Step title="Three letters per word" kind="interactive">

The code reads the bases in groups of **three**. Each triplet — called a **codon** — spells one amino acid. Build a codon and look it up.

<CodonWheel />

Why three? With four bases, pairs give only 16 combinations — too few for 20 amino acids. Triplets give **64** — more than enough.

</Step>

<Step title="Start, stop, and spares" kind="explain">

Those 64 codons are put to clever use:

- **AUG** is the **START** codon — it marks where a protein begins (and codes for the amino acid methionine).
- Three codons (UAA, UAG, UGA) are **STOP** signals — they mark the end, like a full stop.
- The remaining 61 code for the 20 amino acids, so **most amino acids have several codons**. This redundancy is a safety net: many single-letter mistakes still produce the right amino acid.

And it's nearly **universal** — the same codons mean the same amino acids in almost every living thing, from bacteria to humans. That shared code is powerful evidence that all life is related.

<Callout type="info" title="A nearly universal language">
Because the code is shared, a human gene put into a bacterium is read correctly — the basis of genetic engineering, like making bacteria produce human insulin.
</Callout>

</Step>

<Step title="Recap" kind="recap">

- The **genetic code** translates DNA bases into amino acids.
- It reads bases in triplets called **codons** — 64 possible, enough for 20 amino acids.
- **AUG** = start; **UAA/UAG/UGA** = stop; most amino acids have several codons (redundancy).
- The code is nearly **universal**, strong evidence that all life shares one ancestor.

</Step>
