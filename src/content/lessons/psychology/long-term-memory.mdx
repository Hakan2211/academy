<Step title="Not one warehouse but many" kind="explain">

It's tempting to imagine long-term memory as a single vast warehouse where everything you know is stacked together. The truth is far more interesting: it is **several distinct systems**, each storing a different kind of knowledge, each running on its own machinery — and, remarkably, each able to be **damaged independently** of the others.

The proof came from brain injury. Patients have lost the ability to form new conscious memories of events while keeping the ability to learn new *skills* perfectly intact — practising a task they swear they've never done before, yet getting visibly better at it. If memory were one system, that would be impossible. So the map of long-term memory is really a **family tree** of separate systems.

</Step>

<Step title="The great divide: explicit vs implicit" kind="explain">

The first and deepest split is by whether you can *consciously bring the memory to mind*:

- **Explicit (declarative) memory** is knowledge you can deliberately call up and put into words — knowing **that**. The capital of France, what you did last weekend. It feels like "remembering".
- **Implicit (non-declarative) memory** is knowledge expressed through *performance*, without any sense of recall — knowing **how**. Riding a bike, a gut flinch at a once-painful sound. It feels like "just doing".

The eerie thing about implicit memory is that it works invisibly. You can be *shaped* by an experience you cannot consciously remember at all — your behaviour carries a memory your awareness has lost.

</Step>

<Step title="Explore the family tree" kind="interactive">

Each branch splits further. Tap through the tree to see what lives where — and a vivid example of each.

<MemoryTypes />

The branches to know: under **explicit**, *semantic* memory holds general facts (Paris is a capital) while *episodic* memory holds personally lived events (your last birthday) — a kind of mental time travel. Under **implicit**, *procedural* memory holds skills (cycling), alongside *conditioning* (learned emotional associations) and *priming* (a recent exposure speeding later processing). Distinct systems, distinct jobs.

</Step>

<Step title="It's not what you do, it's how deeply you process it" kind="explain">

What decides whether something *reaches* this lasting store in the first place? Not how long you stare at it, but **how deeply you process it** — the **levels of processing** idea from Craik and Lockhart.

- **Shallow (structural)** processing notes surface features — *is the word in capitals?* It barely sticks.
- **Intermediate (phonological)** processing notes sound — *does it rhyme with "cat"?* A bit better.
- **Deep (semantic)** processing engages **meaning** — *does this word fit the sentence? what does it remind me of?* This produces by far the strongest, most durable memories.

The lesson is blunt: memory is a by-product of *thinking about meaning*. Rote repetition (shallow) is weak; **elaboration** — tying the new idea to things you already know — is what carves a deep, easy-to-find trace.

</Step>

<Step title="Two systems, one curve" kind="interactive">

Long-term and short-term memory aren't rivals; they work together, and you can catch both in a single experiment. Study a list of words, then recall as many as you can — and watch the pattern of what survives.

<SerialPosition />

The recall curve bends into a **U**. The **first** words are remembered well (the *primacy effect*) because you had time to rehearse them into **long-term** memory. The **last** words are remembered well (the *recency effect*) because they are still sitting in **short-term** memory at the moment of recall. The middle, rehearsed least and already pushed out, sags. One simple list, and both memory stores reveal themselves at once.

</Step>

<Step title="Recap" kind="recap">

- Long-term memory is **several independent systems**, shown by patients who lose one while keeping another.
- The deepest split: **explicit (declarative)** memory — conscious, "knowing *that*" — versus **implicit (non-declarative)** memory — performance-based, "knowing *how*".
- Explicit divides into **semantic** (facts) and **episodic** (lived events); implicit includes **procedural** (skills), **conditioning**, and **priming**.
- **Levels of processing**: **deep, semantic** (meaning-based) processing builds far stronger memories than shallow rote repetition — so **elaborate**, don't just repeat.
- The **serial-position curve** (a U) reveals both stores at once: **primacy** (long-term, rehearsed) and **recency** (still in short-term).

</Step>
